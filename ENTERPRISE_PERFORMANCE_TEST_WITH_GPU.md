## Методология тестирования с GPU

### 1. Традиционный поиск (grep)
```bash

```

### 2. Векторный поиск (ChromaDB) с GPU
```bash

time python3 chroma_simple_server.py --index --project-root /path/to/project --gpu --gpu-device mps

# Поиск с GPU ускорением
time python3 chroma_simple_server.py --search "query" --results 10 --project-root /path/to/project --gpu --gpu-device mps

# Поиск с CPU (для сравнения)
time python3 chroma_simple_server.py --search "query" --results 10 --project-root /path/to/project
```

## Технические детали реализации GPU

### Поддерживаемые устройства:
1. **CUDA** (NVIDIA GPUs) - автоматическое определение
2. **MPS** (Apple Silicon) - автоматическое определение  
3. **CPU** - fallback при отсутствии GPU

### Конфигурация GPU:
```python
from chroma_simple_server import GPUConfig

gpu_config = GPUConfig(
    enabled=True,
    device="auto",  # auto-detect
    batch_size=32,
    use_mixed_precision=True,
    cache_size=1000
)
```

### Командная строка:
```bash
--gpu                    # Включить GPU ускорение
--gpu-device auto        # Устройство (auto/cuda/cpu/mps)
--gpu-batch-size 32      # Размер батча
--gpu-mixed-precision    # Включить mixed precision
--gpu-cache-size 1000    # Размер кэша
```

## Рекомендации для enterprise-проектов с GPU

### Когда использовать GPU ускорение:

1. **Индексация больших codebases** (> 10,000 файлов)
2. **Batch обработка** множества запросов
3. **Real-time поиск** с требованием низкой задержки
4. **Серверные развертывания** с доступом к GPU

### Конфигурация для разных сценариев:

#### Development (локальная машина):
```bash
# Автоматическое определение лучшего устройства
python chroma_simple_server.py --server --gpu --gpu-device auto
```

#### Production (NVIDIA сервер):
```bash
# CUDA с оптимизированными параметрами
python chroma_simple_server.py --server --gpu --gpu-device cuda --gpu-batch-size 64 --gpu-mixed-precision
```

#### Apple Silicon разработка:
```bash
# MPS с mixed precision (где поддерживается)
python chroma_simple_server.py --server --gpu --gpu-device mps --gpu-batch-size 32
```

## Ограничения и рекомендации

### Ограничения GPU ускорения:
1. **Требует PyTorch с GPU поддержкой**
2. **Зависит от доступной VRAM** (1-2GB для модели)
3. **Mixed precision не всегда поддерживается** (особенно на MPS)
4. **Первая инференция медленнее** из-за компиляции

### Рекомендации по оптимизации:
1. **Начинать с batch size 32**, увеличивать при наличии VRAM
2. **Включать mixed precision** для совместимых GPU
3. **Использовать кэширование** для повторяющихся запросов
4. **Мониторить использование памяти** GPU
