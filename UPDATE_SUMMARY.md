### 3. Ключевые обновления в документации:

#### Производительность GPU:
- **Поисковые запросы:** 2.6x-14.7x ускорение
- **Batch encoding:** до 12.6x ускорение (64 текста)
- **Поддержка устройств:** CUDA (NVIDIA), MPS (Apple Silicon), CPU

#### Конфигурация GPU:
```bash
--gpu                    # Включить GPU ускорение
--gpu-device auto        # Автоопределение устройства
--gpu-batch-size 32      # Размер батча (по умолчанию: 32)
--gpu-mixed-precision    # Включить mixed precision
--gpu-cache-size 1000    # Размер кэша эмбеддингов
```

#### Примеры использования:
```bash
# Автоопределение лучшего устройства
python chroma_simple_server.py --server --gpu

# Специфичное устройство
python chroma_simple_server.py --server --gpu --gpu-device cuda
python chroma_simple_server.py --server --gpu --gpu-device mps

# Оптимизация для batch processing
python chroma_simple_server.py --server --gpu --gpu-batch-size 64 --gpu-mixed-precision
```
